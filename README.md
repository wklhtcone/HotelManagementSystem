# HotelManagementSystem 酒店管理系统
这是一个酒店管理系统，为用户、酒店前台、酒店经理提供服务
功能和配置介绍如下：
## 功能
### 用户端
- 根据时间段查询可用的房间及其信息(房间类型、价格)
- 房间预约
- 查询预约
- 取消预约
- 请求续住
### 酒店前台
- 查看所有房间状态
- 办理直接入住
- 办理预约转入住
- 办理续住
- 办理退房
### 酒店经理端
- 查询、修改房间信息
- 查询、修改员工信息
- 查询日志信息
## 开发环境
- 项目运行平台：`Windows 8/8.1/10`
- 项目开发平台：`Qt5.9 + MinGW` 
- 数据库：`Microsoft SQL Server 2017`
